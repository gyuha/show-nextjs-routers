# Next.js 라우터 표시 CLI 애플리케이션 개발 계획

## 프로젝트 개요
- 이름: `show-nextjs-routers`
- 목적: Next.js 앱의 라우터 구조를 CLI에서 표시
- 기능: 
  - 기본 모드: 라우트 URL 목록 표시
  - 트리 모드(-t): 라우트 계층 구조를 트리 형태로 표시
